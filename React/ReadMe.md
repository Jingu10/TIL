### jsx 문법
* 여러 단어의 조합은 카멜케이스로 작성
* html 태그 안 class 속성은 className 으로 작성
* html 태그 안 for 속성은 htmlFor 으로 작성
* onblur, onfocus, onmousedown 소문자로 작성했던 것들 카멜케이스로 작성하
